Install instructions

Click to download
  1. If you don't have it already, you'll need to get Flatpak.
  2. For Flatpak versions < 0.10.0 it is also recommended to add the flathub repository first. You can download the repository file and use your distribution's software manager (GNOME Software, KDE Discover, ...) to install it.
  3. Click on the INSTALL button.
  4. Your distibution's software manager will open automatically and will install the app.
  5. Once installed, applications can be launched from the usual place in your desktop.

Click to download is available on Arch, Debian Stretch (or later), Endless OS, Fedora 25 (or later), Mageia 6 (or later), and OpenSUSE Tumbleweed.

Command line
  1. If you don't have it already, you'll need to get Flatpak.
  2. Run this on the command line: 
    # Only needed first time you use flathub
flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

flatpak install flathub {{this.app.flatpakAppId}}
  3. Once installed, applications can be launched from the usual place in your desktop.

These command line instructions require Flatpak 0.6.13 or newer. For more information on how to update and manage your applications, see the command line tutorial.
